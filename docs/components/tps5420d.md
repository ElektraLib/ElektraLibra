# TPS5420D

## Genel tanim

**TPS5420D**, Texas Instruments tarafindan uretilen, yuksek giris gerilimini daha dusuk ve regule DC cikis gerilimine donusturmek icin kullanilan bir **step-down buck converter** entegresidir.

!!! note "Kisa ozet"
    TPS5420D, ozellikle endustriyel kartlarda, guc kaynaklarinda ve orta guc DC-DC donusum ihtiyaci olan devrelerde kullanilir.

## Temel teknik ozellikler

| Ozellik | Deger |
|---|---|
| Uretici | Texas Instruments |
| Komponent tipi | Step-down buck converter |
| Topoloji | Anahtarlamali DC-DC regulator |
| Maksimum cikis akimi | 2 A |
| Giris gerilimi | 5.5 V - 36 V |
| Anahtarlama frekansi | Yaklasik 500 kHz |
| Paket | SOIC-8 / D paketi |
| Kontrol tipi | Voltage mode PWM |

## Ne ise yarar?

TPS5420D, yuksek DC gerilimi daha dusuk ve sabit bir DC gerilime cevirmek icin kullanilir. Ornegin 24 V giristen 12 V, 5 V veya 3.3 V gibi regule cikislar elde edilebilir.

## Tipik kullanim alanlari

- Endustriyel kontrol kartlari
- PLC ve otomasyon sistemleri
- LCD / LED surucu kartlari
- Telekom guc katlari
- Motor surucu yardimci beslemeleri
- Genel amacli DC-DC guc donusumu

## Pin yapisi

| Pin | Ad | Gorev |
|---|---|---|
| 1 | BOOT | High-side MOSFET surucusu icin bootstrap beslemesi |
| 2 | NC | Baglanti yok |
| 3 | VSENSE | Feedback / geri besleme girisi |
| 4 | ENA | Enable pini, entegreyi aktif eder |
| 5 | GND | Toprak |
| 6 | PH | Anahtarlama cikisi, bobine gider |
| 7 | VIN | Giris besleme pini |
| 8 | VIN | Giris besleme pini |

!!! warning "Tamir notu"
    VIN var ama cikis yoksa ilk kontrol edilecek pinlerden biri **ENA** pinidir. Enable aktif degilse entegre calismaz.

## Calisma prensibi

TPS5420D icinde anahtarlama yapan bir MOSFET ve PWM kontrol devresi bulunur. Entegre, cikis gerilimini feedback pini uzerinden takip eder. Cikis dusmeye baslarsa PWM gorev oranini artirir, cikis yukselirse gorev oranini azaltir.

Basit akis:

```text
VIN -> Ic MOSFET -> PH pini -> Bobin -> Cikis kondansatoru -> VOUT
                         ^
                         |
                      PWM kontrol
```

## Tipik devre elemanlari

TPS5420D etrafinda genellikle su elemanlar bulunur:

| Eleman | Gorev |
|---|---|
| Bobin | Enerji depolama ve akim filtreleme |
| Schottky diyot | Anahtarlama dongusunde serbest dolasim yolu |
| Giris kondansatoru | VIN hattini filtreler |
| Cikis kondansatoru | Cikis gerilimini duzgunlestirir |
| Bootstrap kondansatoru | BOOT-PH arasinda high-side surucu beslemesi saglar |
| Feedback direncleri | Cikis gerilimini ayarlar |

## Ariza tespit rehberi

### 1. VIN kontrolu

- VIN pinlerinde giris gerilimi var mi?
- Giris kondansatorunde kisa devre var mi?
- VIN gerilimi minimum calisma degerinin uzerinde mi?

### 2. ENA pini kontrolu

- ENA pini aktif seviyede mi?
- ENA pini dusukse devre bilincli olarak kapatilmis olabilir.

### 3. PH pini kontrolu

Osiloskop ile PH pininde anahtarlama sinyali gorulmelidir.

- Kare dalga yoksa entegre enable olmuyor olabilir.
- Kare dalga var ama cikis yoksa bobin, diyot veya cikis kondansatoru kontrol edilmelidir.

### 4. VSENSE pini kontrolu

- VSENSE pini feedback noktasidir.
- Bu pin uzerindeki deger, cikis regule edildiginde referans seviyeye yakin olmalidir.
- Feedback direnclerinden biri kopuksa cikis gerilimi hatali olabilir.

### 5. Isinma kontrolu

- Entegre asiri isiniyorsa cikista kisa devre olabilir.
- Bobin doyuma giriyor olabilir.
- Diyot arizali olabilir.
- PCB layout hatali olabilir.

## Sik gorulen arizalar

| Belirti | Olası sebep |
|---|---|
| Cikis yok | VIN yok, ENA pasif, entegre arizali |
| Cikis dusuk | Feedback hatasi, bobin problemi, yuk fazla |
| Entegre isiniyor | Cikis kisa devre, diyot arizasi, bobin doyumu |
| Cikis dalgali | Cikis kondansatoru bozuk, ESR yuksek |
| Calisip duruyor | Asiri akim korumasi veya termal koruma |

## Olcum noktalari

| Nokta | Beklenen durum |
|---|---|
| VIN | Giris gerilimi mevcut olmali |
| ENA | Aktif seviye olmali |
| PH | Anahtarlama dalga sekli gorulmeli |
| BOOT-PH | Bootstrap gerilimi olusmali |
| VSENSE | Feedback referans seviyesine yakin olmali |
| VOUT | Tasarlanan cikis gerilimi |
